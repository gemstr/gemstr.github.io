How is an instance of something is created?

When an instance of something is created/spawned, it is created from blueprint called a class. Detailed inside these blueprints(classes) can be methods and variables that we want our spawns to have/inherit!

So what's an instance in terms of programming in Ruby?

Those variables methods detailed inside the class are the blueprint variables and methods! When we initialize the Phone class and store into an object(see below), Ruby creates instance variables unique to the instance following the class blueprint, and also makes it inherit the methods(therefore called instance methods).

object_container = Phone.new # "Spawing" a new Phone instance and storing it into object_containter object_container.mute # .mute method was included in the new instance because it was a method that was defined in the class blueprint! object_container.release_date # .release_date is an method that returns the instance variable @release_date
