Cheat Sheet

Cheat Sheet of SQL Query Commands!

SELECT * FROM table;
Selects ALL column data from table
SELECT * FROM Students;
SELECT column1, column 2, .. FROM table;
Selects data from columns specified from table
SELECT LastName, FirstName FROM Students;
SELECT column1, column 2,...
FROM table
[WHERE conditions];
Selects the specified columns from table located where the condition evaluates TRUE.
SELECT LastName, FirstName
FROM Students
WHERE StudentID LIKE '%substring%';
SELECT column1, column2,..
FROM table
[ORDER BY column(s)[DESC/ASC]]
Selects from column1 and column 2 from Table, and orders in Descending or Ascending order.
SELECT LastName, FirstName
FROM table
[ORDER BY column(s)[DESC]];
SELECT column1, column2, ..
FROM table1, table2..
[WHERE condition(s)];
Joins column1 of table 1 and column2 of table2.
SELECT LastName,Points
FROM Students, Assignments
WHERE AssignmentsID = 12 AND
Students.StudentID=Assignments.StudentID;
